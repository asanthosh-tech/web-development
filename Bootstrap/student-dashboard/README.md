# Bootstrap Student Dashboard

A simple Student Dashboard UI built using Bootstrap 5.

## Features

- Responsive Navbar
- Success Alert
- Student Input Form
- Student Cards
- Student Table
- Progress Bar
- Badge Component
- Bootstrap Styling Only

- HTML5
- Bootstrap 5

- Cards
- Typography
- Tables
- Progress Bars
- Badges
