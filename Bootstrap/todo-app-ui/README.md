Todo App UI (Bootstrap)

Description

A responsive Todo App user interface built using Bootstrap 5. The application allows users to add tasks, delete tasks, clear all tasks, and stores data using Local Storage.

Features

- Responsive Bootstrap layout
- Navbar
- Styled input fields
- Task description textarea
- Add task
- Delete individual task
- Clear all tasks
- Local Storage support
- Automatic loading of saved tasks

Technologies Used

- HTML
- Bootstrap 5
- JavaScript
- Local Storage API

Bootstrap Components Used

- Navbar
- Container
- Form Controls
- Buttons
- Grid System
- Tables
- Spacing Utilities


Learning Outcome

This project helped practice building responsive user interfaces with Bootstrap while integrating JavaScript and Local Storage for persistent task management.
