# College Dashboard Portal (Bootstrap Practice)

A responsive College Dashboard UI built using Bootstrap 5.

## Features

- Responsive Navbar
- Hero Section
- Offcanvas Navigation Menu
- Statistics Dashboard Cards
- Responsive Student Table
- Pagination
- Student Search UI
- New Admission Form
- Bootstrap Alerts
- Login Modal
- Delete Confirmation Modal
- Bootstrap Icons
- Contact Section
- Responsive Layout

## Technologies Used

- HTML5
- Bootstrap 5
- Bootstrap Icons

## Note

This project was created for Bootstrap practice only.
It focuses on learning Bootstrap components and responsive layouts.
No JavaScript functionality or backend is included.
