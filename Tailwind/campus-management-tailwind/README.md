# Smart Campus Management Portal

A responsive Smart Campus Management Portal UI built using Tailwind CSS.

## Features

- Responsive Navigation Bar
- Hero Section
- Statistics Dashboard
- Student Registration Form
- Student Data Table
- Course Cards
- Gallery Section
- Latest Notices
- FAQ Section
- Contact Section
- Responsive Footer

## Technologies Used

- HTML5
- Tailwind CSS

## Tailwind Concepts Used

- Flexbox
- Grid
- Responsive Design
- Typography
- Colors
- Borders
- Rounded Corners
- Shadows
- Hover Effects
- Active & Focus States
- Transitions
- Position Utilities
- Forms
- Tables
- Lists
- Animations

## Note

This project was built as a complete Tailwind CSS practice project to learn and apply most of the core Tailwind utilities. It focuses on UI design only and does not include JavaScript functionality or a backend.
