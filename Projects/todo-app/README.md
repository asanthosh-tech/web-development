Todo App

Description

A simple Todo application that allows users to add tasks, delete tasks, and store data using Local Storage so tasks remain available after page refresh.

Features

- Add new tasks
- Delete the last task
- Delete a specific task
- Display total task count
- Store tasks using Local Storage
- Automatically load saved tasks after refresh

Technologies Used

- HTML
- JavaScript
- Local Storage API

Project Flow

Add Task
→ Create Task Object
→ Save to Array
→ Save to Local Storage
→ Display Tasks

Delete Task
→ Remove Task from Array
→ Update Local Storage
→ Refresh Display

Page Refresh
→ Load Local Storage
→ Convert Data Using JSON.parse()
→ Display Saved Tasks

Learning Outcome

This project helped practice DOM manipulation, Local Storage, JSON conversion, dynamic list rendering, event handling, and task management using JavaScript.
