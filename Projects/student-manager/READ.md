Student Manager

Description

A simple JavaScript project to manage students. Users can add students, view the total student count, display passed students, and remove the last added student.

Features

- Add new students
- Generate random marks for each student
- Display total student count
- Show passed students
- Remove the last student
- Dynamic DOM updates

Technologies Used

- HTML
- JavaScript

Concepts Practiced

- Arrays
- Objects
- DOM Manipulation
- Event Listeners
- createElement()
- appendChild()
- filter()
- Functions

How to Run

1. Download the project.
2. Open "index.html" in a browser.
3. Add student names.
4. Click buttons to manage students.

Future Improvements

- Add custom marks input
- Remove specific students
- Edit student details
- Save data using Local Storage
