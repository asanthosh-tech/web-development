Student Manager V2

Description

A student management application that allows users to add, remove, and clear students. Student data is stored using Local Storage, so the data remains available even after refreshing the page.

Features

- Add students
- Remove the last added student
- Clear all students
- Display total student count
- Automatically load saved students on page refresh
- Local Storage support
- Input validation

Technologies Used

- HTML
- CSS
- JavaScript
- Local Storage API


Project Flow

Add Student
→ Update Array
→ Save to Local Storage
→ Display Updated List

Page Refresh
→ Load Local Storage
→ Convert Data Using JSON.parse()
→ Display Saved Students


Learning Outcome

This project helped practice Local Storage, JSON conversion, DOM manipulation, and persistent data handling in JavaScript.
