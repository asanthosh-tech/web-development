Quote Generator

Description

A web application that fetches quotes from an external API and displays a random quote with its author.

Features

- Get random quotes
- Display quote author
- Loading message
- Error handling
- Random quote selection

Technologies Used

- HTML
- CSS
- JavaScript
- Fetch API

Concepts Practiced

- Async/Await
- Fetch API
- JSON Handling
- DOM Manipulation
- Functions
- Random Number Generation
- Error Handling

API Used

https://dummyjson.com/quotes
