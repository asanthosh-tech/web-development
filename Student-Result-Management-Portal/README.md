# Student Result Management Portal

A responsive Student Result Management Portal built using HTML, Tailwind CSS, and Vanilla JavaScript. This project allows users to manage student records with CRUD operations and stores data using the browser's Local Storage.

## Features

- Add Student
- Edit Student
- Delete Student
- Search Student by Name or Roll Number
- Student Cards
- Student Results Table
- Top Students Section
- Live Statistics
  - Total Students
  - Passed Students
  - Failed Students
  - Average CGPA
- Local Storage Support
- Responsive Design
- FAQ Section
- Contact Section
